## 프로젝트 소개

본 사이트는 2024년 성신여자대학교 축제 "수정대동제 Eternal"를 위한 홈페이지입니다. 성신여자대학교 제36대 총학생회 '여일하게'와 성신여자대학교 멋쟁이사자처럼 12기 대동제 TFT팀의 협업으로 제작된 홈페이지입니다.
